# Book Tracker (Трекер прочитанных книг)

**Автор:** Cохроков Алихан
**Вариант:** Book Tracker


---

## Описание программы

Book Tracker — это графическое приложение для управления списком прочитанных книг. Позволяет добавлять книги, удалять их, фильтровать по жанру и количеству страниц. Данные сохраняются в файл books.json.

---

## Требования для запуска

- Python 3.7 или выше
- Библиотеки: tkinter, json

---

## Как запустить

```bash
# Клонировать репозиторий
git clone ..

# Запустить программу
